# Audience in feature flags and feature groups {#audience-overview}

Experience Rollouts provides audience targeting for feature flags and feature groups. You can combine context variables to precisely control which users receive a feature.

For context-based targeting (for example, targeting by the user's active language or device type), see [Use context in audience rules](using-context-in-audience-rules.md).
