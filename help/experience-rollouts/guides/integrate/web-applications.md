# Web applications {#web-applications}

Web applications integrate with Experience Rollouts by calling the **Feature API V3** to retrieve feature flags and placing conditional logic behind them in your application code. Contact Experience Rollouts support for access to the Feature API V3 reference, including request parameters, response format, and example calls.
